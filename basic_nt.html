Networking Commands - Kali Linux

ifconfig

Displays or configures a network interface.

ifconfig eth0 Displays network interfaces.

ip

Manages network interfaces and routes.

ip addr show Shows IP addresses and routes.

ping

Tests connectivity to a remote host.

ping google.com Checks network connectivity.

netstat

Displays network connections and statistics.

netstat -tuln Shows network connections.

traceroute

Shows the path packets take to a network host.

traceroute google.com Traces network route.

nslookup

Queries DNS to find out domain-related information.

nslookup google.com DNS query tool.

dig

Another tool to query DNS information.

dig google.com Another DNS query tool.

route

Displays or modifies the IP routing table.

route -n Shows IP routing table.

nmap

Network scanner used to discover hosts and services.

nmap -sP 192.168.1.0/24 Network scanning tool.

tcpdump

Captures network packets for analysis.

tcpdump -i eth0 Network packet analyzer.

iptables

Configures the Linux kernel firewall.

iptables -L Configures firewall rules.

iwconfig

Displays or configures wireless network interfaces.

iwconfig wlan0 Configures wireless interfaces.

nmcli

Manages NetworkManager via command line.

nmcli dev status NetworkManager CLI tool.

hostname

Displays or sets the system's hostname.

hostname -I Shows system hostname.

whois

Retrieves information about a domain.

whois google.com Domain information tool.

wget

Downloads files from the web.

wget http://example.com/file.zip File downloader.

curl

Transfers data from or to a server.

curl http://example.com Data transfer tool.
